سياسة الخصوصية لتطبيق Core School

مرحباً بك في تطبيق Core School، المطوَّر من قِبل Smart Core. تصف هذه السياسة كيفية جمع البيانات واستخدامها وحمايتها عند استخدامك للتطبيق.

أولاً: البيانات التي قد يجمعها التطبيق

يجمع التطبيق أو يعالج البيانات التالية بحسب دور المستخدم وصلاحياته:

  • اسم ولي الأمر أو المعلم ومعرّف المستخدم.
  • اسم الطالب ومعرّفه.
  • سجلات الحضور والغياب.
  • الدرجات والامتحانات والواجبات المنزلية.
  • الرسائل بين المدرسة وأولياء الأمور والمعلمين.
  • طلبات الإجازة والشكاوى والمرفقات ذات الصلة.
  • الصور أو الملفات التي يرفعها المستخدم عند استخدام ميزات المرفقات.
  • معلومات الرسوم الدراسية المعروضة داخل التطبيق.
  • رمز Firebase Cloud Messaging لإرسال الإشعارات الفورية.
  • نوع الجهاز ونظام التشغيل وإصدار التطبيق لأغراض تشغيل الخدمة.

ثانياً: كيف نستخدم البيانات

تُستخدم البيانات حصراً لتشغيل نظام إدارة المدرسة، بما يشمل:

  • تسجيل دخول المستخدم والتحقق من صلاحياته.
  • عرض معلومات الطالب وفق صلاحيات ولي الأمر أو المعلم.
  • إدارة الحضور والغياب والدرجات والواجبات والامتحانات.
  • تمكين التواصل بين المدرسة وأولياء الأمور والمعلمين.
  • إرسال الإشعارات المتعلقة بالمدرسة.
  • تحسين أمان التطبيق وموثوقيته.

ثالثاً: مشاركة البيانات

  • لا تبيع Smart Core بيانات المستخدمين.
  • لا تشارك البيانات مع أطراف إعلانية أو تسويقية.
  • لا تُستخدم البيانات للإعلانات أو تتبع الإعلانات.
  • قد يُستخدم Firebase Cloud Messaging فقط لإرسال الإشعارات الفورية.

رابعاً: البيانات التي لا يجمعها التطبيق

لا يجمع التطبيق أياً مما يلي:

  • بيانات الموقع الجغرافي.
  • جهات الاتصال.
  • تسجيلات صوتية أو بيانات الميكروفون.
  • البيانات الصحية.
  • بيانات البطاقات البنكية أو الحسابات المصرفية.
  • أي بيانات لأغراض الإعلانات أو تتبع الإعلانات.

خامساً: أذونات التطبيق

  • الكاميرا أو مكتبة الصور: قد تُستخدم عند رفع مرفقات الواجبات أو طلبات الإجازة.
  • الإشعارات: قد تُستخدم لإرسال تنبيهات مدرسية مهمة.

سادساً: أمان البيانات

  • تتم الاتصالات بين التطبيق والخادم عبر بروتوكول HTTPS المشفّر.
  • تُخزَّن رموز الجلسة وبيانات تسجيل الدخول باستخدام التخزين الآمن حيثما أمكن.
  • يعتمد الوصول إلى البيانات على صلاحيات المستخدم داخل النظام.

سابعاً: حذف البيانات وطلبات الخصوصية

يحق للمستخدم أو المدرسة طلب تصحيح البيانات أو حذفها وفق سياسات المدرسة والنظام المعمول بها. لأي استفسار بشأن الخصوصية أو طلب حذف البيانات، يرجى التواصل معنا:

Privacy Policy for Core School

Welcome to Core School, developed by Smart Core. This policy describes how data is collected, used, and protected when you use the application.

1. Data the App May Collect or Process

The app collects or processes the following data depending on the user's role and permissions:

  • Parent or teacher name and user ID.
  • Student name and student ID.
  • Attendance records.
  • Grades, exams, and homework assignments.
  • Messages between the school, parents, and teachers.
  • Leave requests, complaints, and related attachments.
  • Photos or files uploaded by the user when using attachment features.
  • School fee information displayed inside the app.
  • Firebase Cloud Messaging token for push notifications.
  • Device type, operating system, and app version for service operation.

2. How We Use Data

Data is used only for operating the school management system, including:

  • User login and authorization.
  • Displaying student information according to parent or teacher permissions.
  • Managing attendance, grades, homework, and exams.
  • Enabling communication between the school, parents, and teachers.
  • Sending school-related notifications.
  • Improving app security and reliability.

3. Data Sharing

  • Smart Core does not sell user data.
  • Data is not shared with advertising or marketing parties.
  • Data is not used for advertising or ad tracking.
  • Firebase Cloud Messaging may be used only for push notifications.

4. Data the App Does Not Collect

The app does not collect any of the following:

  • Location data.
  • Contacts.
  • Audio or microphone recordings.
  • Health data.
  • Payment card or banking data.
  • Data for advertising or ad tracking.

5. App Permissions

  • Camera or photo library: may be used when uploading homework or leave request attachments.
  • Notifications: may be used to send important school alerts.

6. Data Security

  • Communication between the app and server is protected using HTTPS.
  • Session tokens and login data are stored using secure storage where possible.
  • Access to data depends on user permissions inside the system.

7. Data Deletion and Privacy Requests

The user or school may request data correction or deletion according to the applicable school and system policies. For privacy inquiries and data deletion requests, please contact us:
